# TikiDeco Site V2

Premium 3D static site for the TikiDeco / TIDE Ethereum Sepolia prototype.

The site is intentionally read-only. It does not connect wallets, request signatures, sell tokens, show price charts, or provide transaction flows.

## Visual Concept

TikiDeco V2 uses a restrained luxury-tech direction: deep navy, black graphite, cyan glow, violet accents, glass panels, an abstract night-ocean grid, a minimal resort silhouette, and small blockchain nodes above the water. The scene is symbolic concept visualization, not a completed property.

On mobile or reduced-motion environments, the 3D scene degrades into a lightweight static visual layer.

## Commands

From the repository root:

```bash
npm install
npm run site:dev
npm run site:build
npm run site:preview
npm run site:check
```

The Vite build writes to `site/` so GitHub Pages can continue publishing the existing Pages directory. Canonical deployment data remains sourced from `deployments/canonical.json`.

## Content Boundaries

Public copy must preserve these facts:

- TikiDeco / TIDE is an Ethereum Sepolia testnet prototype.
- TIDE is not offered for sale.
- TIDE has no stated monetary value.
- There is no mainnet deployment.
- Independent audit has not started.
- Hospitality functionality is conceptual or planned, not a live guaranteed benefit.
