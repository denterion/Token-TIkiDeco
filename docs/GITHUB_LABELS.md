# GitHub Labels

Recommended labels for the TikiDeco community preview workflow.

TikiDeco / TIDE is a Sepolia prototype only. TIDE is not offered for sale, has no stated monetary value, is not deployed on mainnet, has no active guest benefits, and has not started an independent audit.

| Label | Purpose |
| --- | --- |
| `v0.2-utility-pilot` | Work related to the v0.2 read-only utility pilot release candidate. |
| `community-preview` | Public feedback collection and preview operations. |
| `site` | Public website UX, accessibility, links, copy, or layout. |
| `documentation` | Documentation gaps, stale facts, unclear structure, or wording fixes. |
| `audit-readiness` | V2 external audit package preparation without claiming audit completion. |
| `claims-check` | Public wording, prohibited claim, or consistency checker work. |
| `legal-review-needed` | Requires counsel review before public or operational use. |
| `privacy-review-needed` | Requires privacy/data-minimization review before use. |
| `operations-gate` | Staff process, support, inventory, dispute, or hospitality operations gate. |
| `testnet-only` | Sepolia-only test flow or public testnet artifact. |
| `good-first-issue` | Narrow contribution that is safe for new contributors. |
| `release-blocker` | Blocks a release candidate or publication checklist. |
| `security-question` | Public, non-sensitive security question. Sensitive reports follow `SECURITY.md`. |
| `translation` | EN/ES/RU localization review. |

Do not use labels or issue titles to imply sale, price, mainnet readiness, active guest benefits, independent audit completion, property rights, or financial return.
