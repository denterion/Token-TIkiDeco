## Extended LinkedIn Post

TikiDeco / TIDE is an open-source Ethereum Sepolia prototype exploring how a hospitality-linked project can publish transparent token rules, allocation planning, vesting records, governance controls, and report hashes.

The current release is intentionally limited and reviewable:

- TIDE exists on Ethereum Sepolia, not mainnet.
- The token and vesting vault source code are verified on Sepolia Etherscan.
- Privileged operations are controlled by a Sepolia Safe.
- Public reports can be anchored through on-chain hashes.
- The website includes a read-only trust dashboard with no wallet connection or transactions.
- OpenZeppelin V2 code exists as candidate review work, not as the active deployment.

This is not a token sale. TIDE currently has no stated monetary value and is not offered for sale. The project does not claim hotel ownership rights, revenue rights, exchange listings, active guest benefits, or an independent audit.

The purpose of the prototype is to make the project state inspectable: what exists, what is planned, what is only conceptual, and what still needs legal, operational, and security review.
