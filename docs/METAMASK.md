# View TIDE In MetaMask

Use this guide to view the Sepolia testnet TIDE token in MetaMask.

## 1. Open MetaMask

Open MetaMask and select the wallet where you want to view the test token.

## 2. Enable Test Networks

If Sepolia is not visible, open MetaMask settings, enable test networks, and select **Ethereum Sepolia**.

## 3. Import Token

In MetaMask, click **Import tokens** and paste the token address:

```text
0xed5dA4F272A6DF8ddB170908108C267563B11621
```

MetaMask should detect:

```text
Symbol: TIDE
Decimals: 18
```

If it does not auto-detect, enter those values manually.

## 4. Important

This is a Sepolia testnet token. It has no market value and is not a mainnet asset.
