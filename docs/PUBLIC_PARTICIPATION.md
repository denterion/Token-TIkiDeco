# Public Participation

Last verified: 2026-07-10.

TikiDeco is an Ethereum Sepolia prototype. Public participation does not create a token sale, stated monetary value, mainnet deployment, active guest benefit, or independent audit status.

## Current GitHub Settings

| Channel | Status | How to use it |
| --- | --- | --- |
| GitHub Issues | Enabled | Use the repository [issue forms](https://github.com/denterion/Token-TIkiDeco/issues/new/choose) for public, non-sensitive feedback. A GitHub account is required. Blank issues are disabled. |
| GitHub Discussions | Disabled | No Discussions channel is currently available. Use an appropriate issue form instead. |
| Private vulnerability reporting | Enabled | Use [GitHub private vulnerability reporting](https://github.com/denterion/Token-TIkiDeco/security/advisories/new) and follow [`SECURITY.md`](../SECURITY.md). |
| Public security issues | Not allowed for sensitive details | Do not publish exploit details, secrets, or unpatched vulnerabilities in a public issue. |

Repository-level Issues are enabled for the public repository. No repository setting was found that limits issue creation to collaborators, but GitHub authentication, platform controls, and repository moderation still apply.

## Participation Paths

### GitHub Issues

Use [issue forms](https://github.com/denterion/Token-TIkiDeco/issues/new/choose) for reproducible bugs, site feedback, utility-pilot feedback, documentation issues, audit-readiness tasks, and verification questions.

### GitHub Discussions

Discussions are currently unavailable. This status must remain visible until the repository setting is explicitly enabled and moderation rules are published.

### Security Disclosure

Use [private vulnerability reporting](https://github.com/denterion/Token-TIkiDeco/security/advisories/new). Do not submit a private key, seed phrase, password, RPC credential, private guest data, or unrelated sensitive personal data.

### Documentation Feedback

Use the [documentation issue form](https://github.com/denterion/Token-TIkiDeco/issues/new?template=documentation_issue.yml) and follow [`FEEDBACK_GUIDE.md`](FEEDBACK_GUIDE.md).

### Translation Feedback

Use the [translation review issue form](https://github.com/denterion/Token-TIkiDeco/issues/new?template=translation_review.yml). Report the locale, page, original wording, and suggested correction without adding unsupported public claims.

## Data Boundary

Public participation must not collect private keys, seed phrases, passwords, private guest data, wallet-to-person identity mappings, or sensitive personal data. Pilot feedback should remain public, minimal, and aggregate where possible.
