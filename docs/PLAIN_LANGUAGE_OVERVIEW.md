# TikiDeco In Plain Language

Status: public-friendly explanation draft.

TikiDeco / TIDE is a public test project on Ethereum Sepolia, a test network used by developers.

The simple idea:

TikiDeco is exploring whether a token system can make future hospitality and community programs more transparent. For example, a project could publish clear rules, show who controls important actions, record public report hashes, and test future loyalty or access ideas before anything real is launched.

## What TIDE Is Today

TIDE is currently:

- a Sepolia testnet prototype;
- not offered for sale;
- not deployed on mainnet;
- not assigned any stated monetary value;
- controlled through a documented Sepolia Safe for privileged V1 actions;
- documented in public repository files.

## What Sepolia Means

Sepolia is a test network. It is used for development and public verification.

Sepolia tokens are not the same as mainnet assets. They should not be treated as money, an investment, or a purchase.

## What The Project Is Testing

TikiDeco is testing:

- transparent token rules;
- public contract verification;
- vesting and allocation documentation;
- Safe-controlled administration;
- public reports;
- possible future loyalty or access workflows.

## What Is Not Live

TikiDeco / TIDE is not currently:

- a token sale;
- a presale;
- an investment product;
- a mainnet launch;
- an exchange-listed asset;
- a production system with a completed independent audit;
- a confirmed hotel partnership;
- a live hotel benefit or membership program.

## Why This Can Still Be Useful

Many crypto projects ask people to trust claims. TikiDeco is trying to make claims checkable before asking for broader attention.

The useful question is not "what is the price?" The useful question is:

Can a public reader verify what exists, what is planned, and what is not approved yet?
