## Implemented Features

- Fixed-supply TIDE token on Ethereum Sepolia.
- ERC-20-style transfers, approvals, allowances, and `transferFrom`.
- Owner-controlled pause and unpause in V1.
- Two-step ownership transfer.
- Treasury receives the initial supply.
- Vesting vault deployed as canonical Sepolia V1 history.
- On-chain report hash publishing.
- Sepolia Safe controls privileged operations after ownership handover.
- Public deployment manifest.
- Static public site with read-only trust dashboard.
- OpenZeppelin V2 candidate code included for review, not promoted.

- TIDE is not offered for sale and has no stated monetary value.
- V1 is legacy prototype code.
- V2 is candidate code and is not the active deployment.
- Public dashboard depends on public Sepolia RPC availability.
- No active hotel benefits, completed property claim, or confirmed partner claim is included in this release.
- No exchange listing claim.
- Legal, operational, and utility design remain subject to review.
