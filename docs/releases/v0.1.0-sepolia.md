## What Works

- Fixed-supply TIDE token on Ethereum Sepolia.
- ERC-20-style transfers, approvals, allowances, and `transferFrom`.
- Owner-controlled pause and unpause in V1.
- Two-step ownership transfer.
- Treasury receives the initial supply.
- Vesting vault deployed as canonical Sepolia V1 history.
- On-chain report hash publishing.
- Sepolia Safe controls privileged operations after ownership handover.
- Public deployment manifest.
- Static public site with read-only verification surfaces.
- V2 contracts exist as candidate review code only and are not promoted by this release.
