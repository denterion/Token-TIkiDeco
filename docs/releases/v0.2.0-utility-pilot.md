# v0.2.0-utility-pilot Release Notes

Release date: 2026-06-26.

Source commit: `5ed20415b569779f8b00245af8b98b9599f77044`.

Status: public pre-release for a planned Sepolia-only utility pilot flow. This is not a mainnet release, not a token sale, not a value statement, and not an independent audit report. The project is not independently audited.

## Summary

v0.2.0-utility-pilot prepares a read-only Sepolia eligibility flow for the TIDE Loyalty Pilot. The flow lets a user manually enter a wallet address, checks Sepolia chain ID through allowlisted RPC endpoints, reads `balanceOf(wallet)` from the canonical TIDE token, and explains whether the wallet could enter manual pilot review.
