# TikiDeco FAQ

## ¿Qué es TikiDeco / TIDE?

TikiDeco / TIDE es un prototipo open-source en Ethereum Sepolia que explora infraestructura transparente de loyalty y access para un futuro concepto hospitality.

## ¿TIDE está desplegado en mainnet?

No. El deployment canónico actual está en Ethereum Sepolia, una red de pruebas.

## ¿TIDE se ofrece a la venta?

No. TIDE no se ofrece a la venta y no tiene valor monetario declarado.

## ¿Qué funciona actualmente?

El deployment actual en Sepolia incluye un token ERC-20-style de fixed supply, verified source code, owner-controlled pause logic, publicación de report hashes, vesting vault, privileged operations controladas por Safe y un dashboard público read-only.

## ¿Qué no existe todavía?

No existe mainnet deployment, token sale, exchange listing claim, independent audit, active hotel benefit, confirmed hotel partnership ni completed property claim.

## ¿Quién controla las privileged operations?

El token y vault canónicos de Sepolia V1 están controlados por el owner Safe indicado en el deployment manifest. El threshold actual está documentado como `3-of-3`.

## ¿El proyecto está audited?

No. El repositorio incluye tests y materiales de internal review, pero independent smart-contract audit has not started.

## ¿Qué puede verificar el público?

El público puede verificar el canonical deployment manifest, Sepolia Etherscan source-code pages, repository docs, test suite, public reports y read-only dashboard.

## ¿Los hospitality perks están activos hoy?

No. Los escenarios de loyalty, access y hospitality son research directions. No son active guest benefits ni commitments.

## ¿TIDE representa ownership en un hotel?

No. El prototipo actual no proporciona hotel ownership, equity, debt, revenue rights ni otros financial rights.

## ¿Para qué sirve el public preview?

El public preview sirve para que la comunidad revise los documentos, el sitio, las Sepolia addresses, el lenguaje público, los utility scenarios y la security posture antes de cualquier decisión sobre production, mainnet, real-world utility o promoción más amplia.
